# ollama sample with node.js backup and front and

## pre install
  1. install ollama:
  2. pull llama3.2:
```bash
     preinstall.sh
```

### backend
  1. yarn install required packages:
```bash
sh start-backup.sh
```
  2. run with: node server.js
```bash
yarn-add-ollama.sh
```

### frontend
  1. install http-server
```bash
npm-install.sh 
```

  2. run frontend: http-server 
```bash
start-http.sh
```
